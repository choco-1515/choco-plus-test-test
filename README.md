原因
InvidiousがYouTubeから取得するストリームURLには2種類あります：

pot=（Proof of Token）パラメータがないURL → Android VRクライアント用で直接再生可能
pot=パラメータがあるURL → WEBブラウザのセッションに紐付いたトークンで、動画プレイヤーに渡すと弾かれる
対策
サーバー側・クライアント側の両方で、pot= を含むURLをストリームリストから除外するようにしました。これにより、再生可能なURLだけがプレイヤーに渡されます。
